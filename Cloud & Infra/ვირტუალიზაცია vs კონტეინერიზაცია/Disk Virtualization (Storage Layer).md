рЃФрЃљрЃџрЃўрЃљрЃю рЃЎрЃљрЃарЃњрЃў ­ЪћЦ  
рЃљрЃ«рЃџрЃљ рЃњрЃљрЃЊрЃљрЃЋрЃљрЃџрЃЌ рЃЏрЃћрЃАрЃљрЃЏрЃћ рЃЊрЃўрЃЊ рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃќрЃћ Рђћ **Disk Virtualization**, рЃљрЃюрЃБ рЃарЃЮрЃњрЃЮрЃа рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА рЃЊрЃўрЃАрЃЎрЃў рЃЋрЃўрЃарЃбрЃБрЃљрЃџрЃБрЃа рЃАрЃљрЃЏрЃДрЃљрЃарЃЮрЃерЃў (VM-рЃћрЃЉрЃерЃў).  
рЃЌрЃБ CPU-рЃА рЃЋрЃДрЃЮрЃцрЃЌ рЃЊрЃарЃЮрЃерЃў рЃЊрЃљ RAM-рЃА рЃАрЃўрЃЋрЃарЃфрЃћрЃерЃў, рЃЊрЃўрЃАрЃЎрЃўрЃА virtualization рЃюрЃўрЃерЃюрЃљрЃЋрЃА, рЃарЃЮрЃЏ **рЃћрЃарЃЌ рЃцрЃўрЃќрЃўрЃЎрЃБрЃа рЃЊрЃўрЃАрЃЎрЃА рЃљрЃю рЃАрЃбрЃЮрЃарЃўрЃ»рЃА рЃБрЃљрЃЏрЃарЃљрЃЋрЃў VM рЃўрЃДрЃћрЃюрЃћрЃЉрЃА рЃЌрЃўрЃЌрЃЦрЃЮрЃА рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃА рЃАрЃљрЃЎрЃБрЃЌрЃљрЃарЃў HDD/SSD рЃ░рЃЦрЃЮрЃюрЃЊрЃћрЃА.**

---

## ­ЪДа 1№ИЈРЃБ рЃърЃарЃўрЃюрЃфрЃўрЃърЃў

рЃарЃЮрЃфрЃљ VM рЃЦрЃЏрЃюрЃў, рЃ░рЃўрЃърЃћрЃарЃЋрЃўрЃќрЃЮрЃарЃў рЃЏрЃљрЃА рЃљрЃФрЃџрЃћрЃЋрЃА РђърЃЊрЃўрЃАрЃЎрЃАРђю Рђћ рЃЏрЃљрЃњрЃарЃљрЃЏ рЃћрЃА рЃЊрЃўрЃАрЃЎрЃў рЃарЃћрЃљрЃџрЃБрЃарЃљрЃЊ:

- рЃЕрЃЋрЃћрЃБрЃџрЃћрЃЉрЃарЃўрЃЋрЃў **рЃцрЃљрЃўрЃџрЃўрЃљ** рЃ░рЃЮрЃАрЃб рЃАрЃўрЃАрЃбрЃћрЃЏрЃљрЃќрЃћ (`.vmdk`, `.vhdx`, `.qcow2`)
    
- рЃљрЃю рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџрЃў рЃЉрЃџрЃЮрЃЎрЃБрЃарЃў рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃўрЃА (device) **рЃћрЃЏрЃБрЃџрЃљрЃфрЃўрЃљрЃљ**
    

VM-рЃўрЃА рЃерЃўрЃњрЃюрЃўрЃЌ OS рЃ«рЃћрЃЊрЃљрЃЋрЃА:

```
/dev/sda  (Linux)
C:\        (Windows)
```

рЃЏрЃљрЃњрЃарЃљрЃЏ рЃАрЃўрЃюрЃљрЃЏрЃЊрЃЋрЃўрЃџрЃћрЃерЃў рЃћрЃА рЃБрЃЉрЃарЃљрЃџрЃЮрЃЊ рЃцрЃљрЃўрЃџрЃўрЃљ рЃ░рЃЮрЃАрЃбрЃќрЃћ.

---

## РџЎ№ИЈ 2№ИЈРЃБ рЃарЃЮрЃњрЃЮрЃа рЃљрЃ«рЃЊрЃћрЃюрЃА рЃ░рЃўрЃърЃћрЃарЃЋрЃўрЃќрЃЮрЃарЃў рЃћрЃЏрЃБрЃџрЃљрЃфрЃўрЃљрЃА

### VM рЃ«рЃћрЃЊрЃљрЃЋрЃА:

- Disk Controller (IDE, SCSI, NVMe, VirtIO)
    
- Disk Device (sda, vda, etc.)
    

### Hypervisor рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃА:

- IO рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃћрЃЉрЃўрЃА (read/write, seek) рЃЏрЃўрЃдрЃћрЃЉрЃљрЃА
    
- рЃљрЃЏ рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃћрЃЉрЃўрЃА рЃњрЃљрЃЊрЃљрЃДрЃЋрЃљрЃюрЃљрЃА рЃцрЃљрЃўрЃџрЃБрЃа рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃљрЃЊ рЃ░рЃЮрЃАрЃбрЃќрЃћ
    
- caching / buffering рЃЮрЃърЃбрЃўрЃЏрЃўрЃќрЃљрЃфрЃўрЃљрЃА
    

> рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ, рЃарЃЮрЃфрЃљ VM рЃгрЃћрЃарЃА рЃцрЃљрЃўрЃџрЃА, рЃ░рЃўрЃърЃћрЃарЃЋрЃўрЃќрЃЮрЃарЃў рЃљрЃАрЃарЃБрЃџрЃћрЃЉрЃА `write()`-рЃА рЃ░рЃЮрЃАрЃбрЃќрЃћ рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃА `.vmdk` рЃцрЃљрЃўрЃџрЃерЃў.

---

## ­ЪњЙ 3№ИЈРЃБ Virtual Disk рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃцрЃЮрЃарЃЏрЃљрЃбрЃћрЃЉрЃў

|рЃцрЃЮрЃарЃЏрЃљрЃбрЃў|Hypervisor|рЃЏрЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃџрЃћрЃЉрЃў|
|---|---|---|
|**VMDK**|VMware|рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃў VMware-рЃерЃў|
|**VHD / VHDX**|Hyper-V|Microsoft-рЃўрЃА рЃцрЃЮрЃарЃЏрЃљрЃбрЃћрЃЉрЃў|
|**QCOW2**|KVM/QEMU|рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃљ snapshot-рЃћрЃЉрЃўрЃА, compression-рЃўрЃА, encryption-рЃўрЃА|
|**RAW**|рЃДрЃЋрЃћрЃџрЃљ|рЃЏрЃљрЃарЃбрЃўрЃЋрЃў, рЃАрЃгрЃарЃљрЃцрЃў, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃЊрЃўрЃЊрЃў рЃќрЃЮрЃЏрЃўрЃА|

рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ:

```
VM1 Рєњ disk1.vmdk (20 GB)
VM2 Рєњ disk2.vmdk (40 GB)
VM3 Рєњ disk3.vmdk (15 GB)
```

рЃДрЃЋрЃћрЃџрЃљ рЃћрЃА рЃцрЃљрЃўрЃџрЃў рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃўрЃюрЃљрЃ«рЃћрЃЉрЃЮрЃЊрЃћрЃА рЃћрЃарЃЌ рЃцрЃўрЃќрЃўрЃЎрЃБрЃа SSD-рЃќрЃћ.

---

## ­ЪДЕ 4№ИЈРЃБ рЃЊрЃўрЃюрЃљрЃЏрЃўрЃБрЃарЃў (thin) vs рЃАрЃбрЃљрЃбрЃўрЃЎрЃБрЃарЃў (thick) рЃЊрЃўрЃАрЃЎрЃћрЃЉрЃў

|рЃбрЃўрЃърЃў|рЃљрЃ«рЃАрЃюрЃљ|рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ|
|---|---|---|
|**Thick Provisioning**|рЃцрЃўрЃќрЃўрЃЎрЃБрЃарЃљрЃЊ рЃўрЃЎрЃљрЃЋрЃћрЃЉрЃА рЃЏрЃЌрЃћрЃџ рЃљрЃЊрЃњрЃўрЃџрЃА рЃЌрЃљрЃЋрЃўрЃЊрЃљрЃюрЃЋрЃћ (рЃЏрЃљрЃњ. 40 GB = 40 GB)|рЃЏрЃљрЃдрЃљрЃџрЃў рЃерЃћрЃАрЃарЃБрЃџрЃћрЃЉрЃљ|
|**Thin Provisioning**|рЃўрЃќрЃарЃЊрЃћрЃЉрЃљ рЃЊрЃўрЃюрЃљрЃЏрЃўрЃЎрЃБрЃарЃљрЃЊ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃарЃЮрЃфрЃљ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃў рЃћрЃЏрЃљрЃбрЃћрЃЉрЃљ|рЃћрЃцрЃћрЃЦрЃбрЃБрЃарЃў рЃАрЃўрЃЋрЃарЃфрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ|

Thin provisioning рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃњрЃљрЃФрЃџрЃћрЃЋрЃА, рЃЊрЃўрЃАрЃЎрЃў РђъovercommitРђю рЃњрЃљрЃ«рЃљрЃЊрЃЮ Рђћ рЃљрЃюрЃБ VM-рЃћрЃЉрЃА рЃ»рЃљрЃЏрЃерЃў рЃЏрЃћрЃбрЃў рЃЏрЃЮрЃфрЃБрЃџрЃЮрЃЉрЃљ рЃ░рЃЦрЃЮрЃюрЃЊрЃћрЃЌ, рЃЋрЃўрЃЊрЃарЃћ рЃарЃћрЃљрЃџрЃБрЃарЃљрЃЊрЃљрЃљ рЃЊрЃўрЃАрЃЎрЃќрЃћ.

---

## ­ЪД░ 5№ИЈРЃБ Snapshot рЃЊрЃљ Clone

рЃ░рЃўрЃърЃћрЃарЃЋрЃўрЃќрЃЮрЃарЃћрЃЉрЃў рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃњрЃљрЃФрЃџрЃћрЃЋрЃћрЃю рЃњрЃљрЃЊрЃљрЃўрЃдрЃЮ **Snapshot**:

- рЃўрЃюрЃљрЃ«рЃљрЃЋрЃА рЃЊрЃўрЃАрЃЎрЃўрЃА рЃЏрЃўрЃЏрЃЊрЃўрЃюрЃљрЃарЃћ рЃЏрЃЊрЃњрЃЮрЃЏрЃљрЃарЃћрЃЮрЃЉрЃљрЃА
    
- рЃЏрЃЮрЃЏрЃљрЃЋрЃљрЃџрЃерЃў рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљ РђърЃЊрЃљрЃЉрЃарЃБрЃюрЃЊрЃћРђю рЃўрЃЏ рЃЊрЃарЃЮрЃА
    

­ЪЊИ Snapshot рЃарЃћрЃљрЃџрЃБрЃарЃљрЃЊ рЃЦрЃЏрЃюрЃўрЃА **delta рЃцрЃљрЃўрЃџрЃА** Рђћ рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃћрЃЉрЃў рЃўрЃюрЃљрЃ«рЃћрЃЉрЃљ рЃфрЃљрЃџрЃЎрЃћ:

```
base.qcow2  Рєљ рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃљрЃЊ рЃАрЃбрЃљрЃбрЃўрЃЎрЃБрЃарЃў
delta1.qcow2 Рєљ рЃерЃћрЃўрЃфрЃљрЃЋрЃА рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃћрЃЉрЃА
```

Clone-рЃўрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў рЃ░рЃўрЃърЃћрЃарЃЋрЃўрЃќрЃЮрЃарЃў рЃБрЃЉрЃарЃљрЃџрЃЮрЃЊ рЃљрЃЎрЃЮрЃърЃўрЃарЃћрЃЉрЃА рЃўрЃЏрЃўрЃ»рЃА рЃЊрЃљ рЃЦрЃЏрЃюрЃўрЃА рЃљрЃ«рЃљрЃџ VM-рЃА рЃўрЃЏрЃљрЃЋрЃћ рЃАрЃљрЃгрЃДрЃўрЃАрЃўрЃЊрЃљрЃю Рђћ рЃћрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃњрЃљрЃФрЃџрЃћрЃЋрЃА рЃАрЃгрЃарЃљрЃцрЃљрЃЊ рЃерЃћрЃЦрЃЏрЃюрЃљ рЃбрЃћрЃАрЃбрЃўрЃА рЃљрЃю dev рЃњрЃљрЃарЃћрЃЏрЃЮ.

---

## РџЎ№ИЈ 6№ИЈРЃБ Disk passthrough (Direct I/O)

рЃќрЃЮрЃњрЃ»рЃћрЃа VM-рЃА рЃАрЃГрЃўрЃарЃЊрЃћрЃЉрЃљ рЃърЃўрЃарЃЊрЃљрЃърЃўрЃарЃў рЃгрЃЋрЃЊрЃЮрЃЏрЃљ рЃЊрЃўрЃАрЃЎрЃќрЃћ (рЃЏрЃљрЃњ. Database Server).

рЃљрЃЏ рЃЊрЃарЃЮрЃА рЃ░рЃўрЃърЃћрЃарЃЋрЃўрЃќрЃЮрЃарЃў bypass-рЃА рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃА:

- рЃЊрЃўрЃАрЃЎрЃў рЃЏрЃўрЃћрЃюрЃўрЃГрЃћрЃЉрЃљ рЃърЃўрЃарЃЊрЃљрЃърЃўрЃа VM-рЃА рЃарЃЮрЃњрЃЮрЃарЃф рЃцрЃўрЃќрЃўрЃЎрЃБрЃарЃў рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃљ (`/dev/sdb`)
    
- рЃ░рЃўрЃърЃћрЃарЃЋрЃўрЃќрЃЮрЃарЃў рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ I/O рЃљрЃарЃ«рЃА рЃљрЃЎрЃЮрЃюрЃбрЃарЃЮрЃџрЃћрЃЉрЃА
    

> рЃљрЃЏ рЃЏрЃўрЃЊрЃњрЃЮрЃЏрЃљрЃА рЃћрЃгрЃЮрЃЊрЃћрЃЉрЃљ **Disk Passthrough** Рђћ рЃњрЃљрЃќрЃарЃЊрЃўрЃџрЃў рЃерЃћрЃАрЃарЃБрЃџрЃћрЃЉрЃљ, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃюрЃљрЃЎрЃџрЃћрЃЉрЃў рЃўрЃќрЃЮрЃџрЃљрЃфрЃўрЃљ.

---

## РџЎ№ИЈ 7№ИЈРЃБ Shared Storage (Network-based)

VM-рЃћрЃЉрЃў рЃ«рЃерЃўрЃарЃљрЃЊ рЃљрЃа рЃўрЃюрЃљрЃ«рЃљрЃЋрЃћрЃю рЃЊрЃўрЃАрЃЎрЃћрЃЉрЃА рЃџрЃЮрЃЎрЃљрЃџрЃБрЃарЃљрЃЊ, рЃљрЃарЃљрЃЏрЃћрЃЊ рЃЦрЃАрЃћрЃџрЃерЃў:

- **iSCSI, NFS, SMB, Ceph, GlusterFS, SAN**
    
- рЃ░рЃўрЃърЃћрЃарЃЋрЃўрЃќрЃЮрЃарЃў рЃўрЃдрЃћрЃЉрЃА рЃЉрЃџрЃЮрЃЎрЃБрЃа рЃљрЃю рЃцрЃљрЃўрЃџрЃБрЃа storage-рЃА рЃЦрЃАрЃћрЃџрЃўрЃЊрЃљрЃю
    
- VM-рЃћрЃЉрЃў рЃ«рЃћрЃЊрЃљрЃЋрЃћрЃю рЃћрЃА рЃЊрЃўрЃАрЃЎрЃў рЃарЃЮрЃњрЃЮрЃарЃф рЃџрЃЮрЃЎрЃљрЃџрЃБрЃарЃў
    

рЃћрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃўрЃФрЃџрЃћрЃЋрЃљ:

- Live Migration (VM рЃњрЃљрЃЊрЃљрЃљрЃЊрЃњрЃўрЃџрЃћрЃЉрЃљ рЃАрЃ«рЃЋрЃљ рЃАрЃћрЃарЃЋрЃћрЃарЃќрЃћ downtime-рЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ)
    
- High Availability (HA)
    
- Backup/restore рЃЏрЃљрЃарЃбрЃўрЃЋрЃљрЃЊ

## ­ЪДЕ 8№ИЈРЃБ Performance Optimization

рЃ░рЃўрЃърЃћрЃарЃЋрЃўрЃќрЃЮрЃарЃћрЃЉрЃў рЃўрЃДрЃћрЃюрЃћрЃЉрЃћрЃю VirtIO / paravirtualized рЃЊрЃарЃљрЃўрЃЋрЃћрЃарЃћрЃЉрЃА:

- Guest OS-рЃерЃў рЃЊрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃЊрЃарЃљрЃўрЃЋрЃћрЃарЃў рЃБрЃерЃБрЃљрЃџрЃЮрЃЊ рЃћрЃАрЃљрЃБрЃЉрЃарЃћрЃЉрЃљ рЃ░рЃўрЃърЃћрЃарЃЋрЃўрЃќрЃЮрЃарЃА (рЃљрЃарЃљ рЃћрЃЏрЃБрЃџрЃўрЃарЃћрЃЉрЃБрЃџ рЃЎрЃЮрЃюрЃбрЃарЃЮрЃџрЃћрЃарЃА)
    
- рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃюрЃљрЃЊ рЃљрЃЏрЃфрЃўрЃарЃћрЃЉрЃА latency-рЃА рЃЊрЃљ CPU utilization-рЃА
    

> рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ:  
> `VirtIO-SCSI` рЃљрЃю `VirtIO-Block` рЃЉрЃћрЃЋрЃарЃљрЃЊ рЃБрЃцрЃарЃЮ рЃАрЃгрЃарЃљрЃцрЃўрЃљ рЃЋрЃўрЃЊрЃарЃћ рЃћрЃЏрЃБрЃџрЃўрЃарЃћрЃЉрЃБрЃџрЃў IDE.

## ­Ъћ╣ 9№ИЈРЃБ рЃЎрЃЮрЃЏрЃЉрЃўрЃюрЃљрЃфрЃўрЃљ RAM-рЃЌрЃљрЃю рЃЊрЃљ CPU-рЃЌрЃљрЃю
