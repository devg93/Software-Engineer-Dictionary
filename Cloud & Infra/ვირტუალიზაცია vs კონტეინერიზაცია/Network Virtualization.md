рЃњрЃћрЃюрЃўрЃљрЃџрЃБрЃарЃўрЃљ ­ЪћЦ  
рЃљрЃ«рЃџрЃљ рЃњрЃљрЃЋрЃљрЃарЃЕрЃўрЃЮрЃЌ рЃЉрЃЮрЃџрЃЮ рЃцрЃБрЃюрЃЊрЃљрЃЏрЃћрЃюрЃбрЃБрЃарЃў рЃцрЃћрЃюрЃљ Рђћ  
­ЪЉЅ **Network Virtualization** Рђћ рЃљрЃюрЃБ рЃарЃЮрЃњрЃЮрЃа рЃўрЃДрЃЮрЃцрЃљ рЃћрЃарЃЌрЃў рЃцрЃўрЃќрЃўрЃЎрЃБрЃарЃў рЃЦрЃАрЃћрЃџрЃўрЃА рЃЉрЃљрЃарЃљрЃЌрЃў (NIC) рЃЏрЃарЃљрЃЋрЃљрЃџ рЃЋрЃўрЃарЃбрЃБрЃљрЃџрЃБрЃа рЃЏрЃљрЃюрЃЦрЃљрЃюрЃљрЃА рЃерЃЮрЃарЃўрЃА рЃўрЃАрЃћ, рЃарЃЮрЃЏ рЃДрЃЋрЃћрЃџрЃљрЃА рЃ░рЃЦрЃЮрЃюрЃЊрЃћрЃА рЃЌрЃљрЃЋрЃўрЃАрЃў IP, MAC рЃЊрЃљ izolated рЃбрЃарЃљрЃцрЃўрЃЎрЃў.

рЃћрЃА рЃљрЃарЃўрЃА рЃЋрЃўрЃарЃбрЃБрЃљрЃџрЃўрЃќрЃљрЃфрЃўрЃўрЃА рЃћрЃЎрЃЮрЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃА рЃЏрЃћрЃЮрЃЌрЃ«рЃћ рЃАрЃљрЃцрЃБрЃФрЃЋрЃћрЃџрЃў CPU + Memory + Disk + Network рЃАрЃбрЃћрЃЎрЃерЃў.

---

## ­ЪДа 1№ИЈРЃБ рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃўрЃЊрЃћрЃљ

рЃцрЃўрЃќрЃўрЃЎрЃБрЃарЃў рЃАрЃћрЃарЃЋрЃћрЃарЃќрЃћ рЃњрЃЋрЃљрЃЦрЃЋрЃА рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ **рЃћрЃарЃЌрЃў рЃљрЃю рЃарЃљрЃЏрЃЊрЃћрЃюрЃўрЃЏрЃћ NIC** (Network Interface Card).  
рЃЏрЃљрЃњрЃарЃљрЃЏ Hypervisor рЃљрЃЏ рЃЉрЃљрЃарЃљрЃЌрЃќрЃћ рЃЦрЃЏрЃюрЃўрЃА **рЃЋрЃўрЃарЃбрЃБрЃљрЃџрЃБрЃа рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃћрЃЉрЃА (vNICs)**,  
рЃЊрЃљ рЃЌрЃўрЃЌрЃЮ VM-рЃА рЃљрЃФрЃџрЃћрЃЋрЃА рЃћрЃарЃЌ рЃљрЃю рЃЏрЃћрЃб рЃљрЃАрЃћрЃЌ рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃА.

```
Physical NIC (eth0)
   РєЊ
Virtual Switch (vSwitch)
   РєЊ
VM1 Рєњ vNIC1 (eth0)
VM2 Рєњ vNIC2 (eth0)
VM3 Рєњ vNIC3 (eth0)
```

­ЪЉЅ рЃњрЃљрЃарЃћрЃЊрЃљрЃю рЃДрЃЋрЃћрЃџрЃљрЃА рЃ░рЃњрЃЮрЃюрЃўрЃљ, рЃарЃЮрЃЏ VM-рЃћрЃЉрЃА рЃАрЃљрЃЎрЃБрЃЌрЃљрЃарЃў рЃЦрЃАрЃћрЃџрЃўрЃА рЃЉрЃљрЃарЃљрЃЌрЃў рЃљрЃЦрЃЋрЃЌ,  
рЃЏрЃљрЃњрЃарЃљрЃЏ рЃАрЃўрЃюрЃљрЃЏрЃЊрЃЋрЃўрЃџрЃћрЃерЃў рЃДрЃЋрЃћрЃџрЃљ рЃбрЃарЃљрЃцрЃўрЃЎрЃў рЃњрЃљрЃЊрЃўрЃА рЃћрЃарЃЌ рЃцрЃўрЃќрЃўрЃЎрЃБрЃа рЃЉрЃљрЃарЃљрЃЌрЃќрЃћ Hypervisor-рЃўрЃА рЃЏрЃћрЃерЃЋрЃћрЃЮрЃЉрЃўрЃЌ.

---

## РџЎ№ИЈ 2№ИЈРЃБ рЃарЃЮрЃњрЃЮрЃа рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА Рђћ Virtual Switch

Hypervisor рЃЦрЃЏрЃюрЃўрЃА рЃерЃўрЃЊрЃљ **рЃЋрЃўрЃарЃбрЃБрЃљрЃџрЃБрЃа рЃАрЃЋрЃўрЃЕрЃА (vSwitch)**  
рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃћрЃЏрЃБрЃџрЃўрЃарЃћрЃЉрЃА Ethernet switch-рЃўрЃА рЃЦрЃфрЃћрЃЋрЃљрЃА.

**vSwitch**:

- рЃБрЃЎрЃљрЃЋрЃерЃўрЃарЃЊрЃћрЃЉрЃљ рЃЌрЃўрЃЌрЃЮ VM-рЃўрЃА vNIC-рЃА
    
- рЃљрЃЎрЃЮрЃюрЃбрЃарЃЮрЃџрЃћрЃЉрЃА рЃбрЃарЃљрЃцрЃўрЃЎрЃўрЃА рЃЏрЃўрЃЏрЃљрЃарЃЌрЃБрЃџрЃћрЃЉрЃљрЃА (MAC-рЃќрЃћ рЃљрЃю VLAN-рЃќрЃћ рЃЊрЃљрЃДрЃарЃЊрЃюрЃЮрЃЉрЃўрЃЌ)
    
- рЃљрЃ«рЃЮрЃарЃфрЃўрЃћрЃџрЃћрЃЉрЃА рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮрЃћрЃЉрЃљрЃА (firewall, isolation, rate limit)
    

### рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃфрЃюрЃЮрЃЉрЃўрЃџрЃў vSwitch-рЃћрЃЉрЃў:

|Hypervisor|Switch рЃбрЃўрЃърЃў|
|---|---|
|VMware ESXi|vSwitch, vDS|
|Hyper-V|vSwitch|
|KVM/Proxmox|Linux Bridge / Open vSwitch|
|Xen|Open vSwitch|

---

## ­ЪДЕ 3№ИЈРЃБ Network рЃбрЃўрЃърЃћрЃЉрЃў VM-рЃўрЃАрЃЌрЃЋрЃўрЃА

### ­Ъћ╣ A. **Bridge Mode (External Network)**

VM рЃБрЃЎрЃљрЃЋрЃерЃўрЃарЃЊрЃћрЃЉрЃљ рЃърЃўрЃарЃЊрЃљрЃърЃўрЃа рЃцрЃўрЃќрЃўрЃЎрЃБрЃа рЃЦрЃАрЃћрЃџрЃА:

- рЃўрЃдрЃћрЃЉрЃА IP-рЃА рЃЌрЃЦрЃЋрЃћрЃюрЃў LAN-рЃўрЃЊрЃљрЃю (рЃЏрЃљрЃњ. DHCP router-рЃўрЃЊрЃљрЃю)
    
- рЃерЃћрЃБрЃФрЃџрЃўрЃљ рЃАрЃ«рЃЋрЃљ рЃЎрЃЮрЃЏрЃърЃўрЃБрЃбрЃћрЃарЃћрЃЉрЃЌрЃљрЃю рЃърЃўрЃарЃЊрЃљрЃърЃўрЃа рЃЎрЃЮрЃЏрЃБрЃюрЃўрЃЎрЃљрЃфрЃўрЃљ
    

```
VM Рєћ vSwitch Рєћ Physical NIC Рєћ Switch Рєћ Network
```

### ­Ъћ╣ B. **NAT Mode (Network Address Translation)**

VM-рЃћрЃЉрЃў рЃўрЃДрЃћрЃюрЃћрЃЉрЃћрЃю рЃћрЃарЃЌ рЃАрЃљрЃћрЃарЃЌрЃЮ рЃњрЃљрЃарЃћ IP-рЃА, рЃарЃЮрЃњрЃЮрЃарЃф рЃ░рЃЮрЃАрЃбрЃў.

- VM Рєњ рЃ░рЃўрЃърЃћрЃарЃЋрЃўрЃќрЃЮрЃарЃў Рєњ рЃўрЃюрЃбрЃћрЃарЃюрЃћрЃбрЃў (NAT gateway)
    
- рЃ░рЃЮрЃАрЃбрЃўрЃЊрЃљрЃю рЃ«рЃћрЃЊрЃљрЃЋ VM-рЃА, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃњрЃљрЃарЃћрЃЊрЃљрЃю Рђћ рЃљрЃарЃљ
    

```
VM Рєћ vSwitch Рєћ NAT Рєћ Physical NIC Рєћ Internet
```

> рЃўрЃЊрЃћрЃљрЃџрЃБрЃарЃўрЃљ рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА рЃљрЃю рЃарЃЮрЃфрЃљ рЃњрЃўрЃюрЃЊрЃљ, рЃарЃЮрЃЏ VM рЃўрЃДрЃЮрЃА рЃЊрЃљрЃфрЃБрЃџрЃў рЃњрЃљрЃарЃћрЃЊрЃљрЃю.

### ­Ъћ╣ C. **Host-Only Network**

VM рЃЊрЃљ рЃ░рЃЮрЃАрЃб OS рЃБрЃЎрЃљрЃЋрЃерЃўрЃарЃЊрЃћрЃЉрЃўрЃљрЃю рЃћрЃарЃЌрЃЏрЃљрЃюрЃћрЃЌрЃА, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃњрЃљрЃарЃћ рЃЦрЃАрЃћрЃџрЃў рЃЏрЃўрЃБрЃгрЃЋрЃЊрЃЮрЃЏрЃћрЃџрЃўрЃљ.  
рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃўрЃќрЃЮрЃџрЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА.

---

## РџЎ№ИЈ 4№ИЈРЃБ MAC / IP рЃЏрЃљрЃарЃЌрЃЋрЃљ

рЃ░рЃўрЃърЃћрЃарЃЋрЃўрЃќрЃЮрЃарЃў рЃЌрЃўрЃЌрЃЮ vNIC-рЃА рЃљрЃФрЃџрЃћрЃЋрЃА:

- рЃБрЃюрЃўрЃЎрЃљрЃџрЃБрЃа **MAC рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃА**
    
- рЃЊрЃљ рЃАрЃљрЃГрЃўрЃарЃЮрЃћрЃЉрЃўрЃАрЃљрЃЏрЃћрЃЉрЃа **рЃАрЃбрЃљрЃбрЃўрЃЎрЃБрЃа рЃљрЃю DHCP IP-рЃА**
    

рЃ░рЃўрЃърЃћрЃарЃЋрЃўрЃќрЃЮрЃарЃў рЃљрЃАрЃћрЃЋрЃћ рЃЏрЃљрЃарЃЌрЃљрЃЋрЃА **ARP рЃбрЃарЃљрЃцрЃўрЃЎрЃА** рЃўрЃАрЃћ, рЃарЃЮрЃњрЃЮрЃарЃф рЃюрЃљрЃЏрЃЊрЃЋрЃўрЃџрЃў switch.

---

## РџЎ№ИЈ 5№ИЈРЃБ VLAN Integration (Tagging / Trunking)

рЃЏрЃљрЃдрЃљрЃџрЃў рЃўрЃќрЃЮрЃџрЃљрЃфрЃўрЃўрЃАрЃЌрЃЋрЃўрЃА VM-рЃћрЃЉрЃўрЃА рЃбрЃарЃљрЃцрЃўрЃЎрЃў рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃЊрЃљрЃўрЃДрЃЮрЃА VLAN-рЃћрЃЉрЃўрЃЌ:

|VLAN ID|рЃАрЃћрЃњрЃЏрЃћрЃюрЃбрЃў|
|---|---|
|10|Management|
|20|Database|
|30|Web Servers|

рЃцрЃўрЃќрЃўрЃЎрЃБрЃарЃў рЃЉрЃљрЃарЃљрЃЌрЃў рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА trunk рЃарЃћрЃЪрЃўрЃЏрЃерЃў рЃЊрЃљ рЃљрЃбрЃљрЃарЃћрЃЉрЃА рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ VLAN рЃбрЃарЃљрЃцрЃўрЃЎрЃА.  
рЃ░рЃўрЃърЃћрЃарЃЋрЃўрЃќрЃЮрЃарЃў рЃБрЃЉрЃарЃљрЃџрЃЮрЃЊ Tag-рЃћрЃЉрЃА рЃљрЃЏрЃљрЃбрЃћрЃЉрЃА рЃљрЃю рЃљрЃерЃЮрЃарЃћрЃЉрЃА рЃЌрЃўрЃЌрЃЮ vNIC-рЃА.

> рЃерЃћрЃЊрЃћрЃњрЃљрЃЊ, VM-рЃћрЃЉрЃў рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ VLAN-рЃќрЃћ рЃЋрЃћрЃа рЃ«рЃћрЃЊрЃљрЃЋрЃћрЃю рЃћрЃарЃЌрЃЏрЃљрЃюрЃћрЃЌрЃА.

---

## ­ЪДЕ 6№ИЈРЃБ SR-IOV рЃЊрЃљ PCI Passthrough (Direct NIC Access)

рЃќрЃБрЃАрЃбрЃљрЃЊ рЃўрЃАрЃћ, рЃарЃЮрЃњрЃЮрЃарЃф рЃЊрЃўрЃАрЃЎрЃќрЃћ passthrough,  
рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљ рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџрЃў NIC рЃЏрЃЌрЃџрЃўрЃљрЃюрЃљрЃЊ рЃњрЃљрЃЊрЃљрЃАрЃфрЃћ VM-рЃА рЃърЃўрЃарЃЊрЃљрЃърЃўрЃа.

рЃЏрЃљрЃњрЃарЃљрЃЏ рЃБрЃцрЃарЃЮ рЃЊрЃљрЃ«рЃЋрЃћрЃгрЃўрЃџрЃў рЃЋрЃћрЃарЃАрЃўрЃљрЃљ:

### **SR-IOV (Single Root I/O Virtualization)**

- NIC рЃўрЃДрЃЮрЃцрЃљ hardware-рЃЊрЃЮрЃюрЃћрЃќрЃћ РђюVirtual FunctionРђЮ-рЃћрЃЉрЃљрЃЊ (VF)
    
- рЃЌрЃўрЃЌрЃЮ VF рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃЏрЃўрЃћрЃюрЃўрЃГрЃЮрЃА рЃњрЃљрЃюрЃАрЃ«рЃЋрЃљрЃЋрЃћрЃЉрЃБрЃџ VM-рЃА
    
- рЃДрЃЋрЃћрЃџрЃљ рЃўрЃДрЃћрЃюрЃћрЃЉрЃА рЃърЃўрЃарЃЊрЃљрЃърЃўрЃа hardware queue-рЃА
    

­ЪЉЅ рЃљрЃЏрЃўрЃЌ latency рЃЌрЃўрЃЌрЃЦрЃЏрЃўрЃА рЃЦрЃарЃћрЃЉрЃљ рЃЊрЃљ I/O throughput рЃўрЃќрЃарЃЊрЃћрЃЉрЃљ.

­ЪЊЇ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ high-performance workload-рЃћрЃЉрЃерЃў (Datacenter, Cloud, Database).

---

## РџЎ№ИЈ 7№ИЈРЃБ Overlay Networking (Cloud-level)

рЃЊрЃўрЃЊ cloud рЃърЃџрЃљрЃбрЃцрЃЮрЃарЃЏрЃћрЃЉрЃерЃў (AWS, Azure, OpenStack, Kubernetes)  
VM-рЃћрЃЉрЃў рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃўрЃДрЃЋрЃюрЃћрЃю рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃАрЃћрЃарЃЋрЃћрЃарЃќрЃћ, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃўрЃЏрЃљрЃЋрЃћ subnet-рЃерЃў РђърЃЋрЃўрЃарЃбрЃБрЃљрЃџрЃБрЃарЃљрЃЊРђю.

рЃљрЃЏрЃўрЃАрЃЌрЃЋрЃўрЃА рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ:

- **VXLAN / GRE tunnels** Рђћ рЃЦрЃЏрЃюрЃўрЃА рЃЋрЃўрЃарЃбрЃБрЃљрЃџрЃБрЃа Overlay рЃЦрЃАрЃћрЃџрЃА
    
- Hypervisor-рЃћрЃЉрЃў рЃћрЃарЃЌрЃЏрЃљрЃюрЃћрЃЌрЃЌрЃљрЃю рЃљрЃњрЃќрЃљрЃЋрЃюрЃўрЃљрЃю encapsulated рЃърЃљрЃЎрЃћрЃбрЃћрЃЉрЃА
    

> рЃерЃћрЃЊрЃћрЃњрЃў: рЃДрЃЋрЃћрЃџрЃљ VM рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА рЃарЃЮрЃњрЃЮрЃарЃф рЃћрЃарЃЌрЃў рЃЊрЃўрЃЊрЃў LAN,  
> рЃЏрЃўрЃБрЃ«рЃћрЃЊрЃљрЃЋрЃљрЃЊ рЃўрЃЏрЃўрЃАрЃљ, рЃарЃЮрЃЏ рЃцрЃўрЃќрЃўрЃЎрЃБрЃарЃљрЃЊ рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃАрЃћрЃарЃЋрЃћрЃарЃќрЃћрЃљ.

## ­ЪДЕ 8№ИЈРЃБ Security Features

Hypervisor-рЃА рЃерЃћрЃБрЃФрЃџрЃўрЃљ:

- Firewall per vNIC
    
- Rate limiting (bandwidth control)
    
- MAC spoofing protection
    
- IP filtering
    
- Promiscuous mode control (network sniffing-рЃўрЃА рЃгрЃўрЃюрЃљрЃљрЃдрЃЏрЃЊрЃћрЃњ)
    

рЃћрЃА рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА, рЃарЃЮрЃЏ рЃћрЃарЃЌрЃў VM рЃЋрЃћрЃа рЃерЃћрЃўрЃГрЃарЃљрЃА рЃЏрЃћрЃЮрЃарЃўрЃА рЃбрЃарЃљрЃцрЃўрЃЎрЃерЃў.
