рЃФрЃљрЃџрЃўрЃљрЃю рЃЎрЃљрЃарЃњрЃў рЃерЃћрЃЎрЃўрЃЌрЃ«рЃЋрЃљрЃљ Рђћ **РђърЃЎрЃћрЃарЃюрЃћрЃџрЃўРђю (Kernel)** рЃљрЃарЃўрЃА рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃБрЃџрЃў рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃА РђърЃњрЃБрЃџрЃўРђю.  
рЃЌрЃБ рЃћрЃА рЃћрЃарЃЌрЃ«рЃћрЃџ рЃАрЃгрЃЮрЃарЃљрЃЊ рЃњрЃљрЃўрЃњрЃћ Рђћ рЃЏрЃћрЃарЃћ рЃЏрЃЌрЃћрЃџрЃў рЃЎрЃЮрЃЏрЃърЃўрЃБрЃбрЃћрЃарЃБрЃџрЃў рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃљ рЃњрЃљрЃњрЃўрЃљрЃЊрЃЋрЃўрЃџрЃЊрЃћрЃЉрЃљ.  
рЃЏрЃЮрЃЊрЃў рЃљрЃњрЃўрЃ«рЃАрЃюрЃў рЃЏрЃљрЃарЃбрЃўрЃЋрЃљрЃЊ, рЃЌрЃљрЃю рЃърЃарЃЮрЃцрЃћрЃАрЃўрЃЮрЃюрЃљрЃџрЃБрЃарЃљрЃЊ ­ЪЉЄ

## ­ЪДа рЃЏрЃЮрЃЎрЃџрЃћрЃЊ:

**Kernel** = рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃБрЃџрЃў рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃА рЃўрЃА рЃюрЃљрЃгрЃўрЃџрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃърЃўрЃарЃЊрЃљрЃърЃўрЃа РђърЃћрЃАрЃљрЃБрЃЉрЃарЃћрЃЉрЃљРђю рЃЎрЃЮрЃЏрЃърЃўрЃБрЃбрЃћрЃарЃўрЃА **рЃ░рЃљрЃарЃЊрЃЋрЃћрЃарЃА** (CPU, RAM, Disk, Network).

рЃДрЃЋрЃћрЃџрЃљ рЃАрЃ«рЃЋрЃљ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ Рђћ рЃЉрЃарЃљрЃБрЃќрЃћрЃарЃў, VS Code, Docker, рЃерЃћрЃюрЃў рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљ Рђћ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃЎрЃћрЃарЃюрЃћрЃџрЃўрЃА рЃЏрЃћрЃерЃЋрЃћрЃЮрЃЉрЃўрЃЌ.

## ­Ъћ╣ рЃарЃљ рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃА рЃЎрЃћрЃарЃюрЃћрЃџрЃў

рЃЎрЃћрЃарЃюрЃћрЃџрЃў **рЃљрЃа рЃљрЃарЃўрЃА рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ**, рЃљрЃарЃљрЃЏрЃћрЃЊ рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЏрЃљрЃарЃЌрЃљрЃЋрЃА рЃДрЃЋрЃћрЃџрЃљрЃцрЃћрЃарЃА рЃЦрЃЋрЃћрЃЏрЃЮрЃЌ:

|рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ|рЃарЃЮрЃџрЃў|
|---|---|
|**Process Management**|рЃЏрЃљрЃарЃЌрЃљрЃЋрЃА рЃърЃарЃЮрЃфрЃћрЃАрЃћрЃЉрЃА (рЃарЃЮрЃЊрЃўрЃА рЃўрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃА CPU-рЃќрЃћ рЃЋрЃўрЃю)|
|**Memory Management**|рЃљрЃюрЃљрЃгрЃўрЃџрЃћрЃЉрЃА RAM-рЃА рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃћрЃЉрЃА рЃерЃЮрЃарЃўрЃА|
|**Device Drivers**|рЃљрЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃА рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃћрЃЉрЃА (disk, keyboard, network)|
|**File System**|рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃгрЃћрЃарЃљ/рЃгрЃљрЃерЃџрЃљ/рЃЊрЃљрЃЎрЃўрЃЌрЃ«рЃЋрЃўрЃА рЃџрЃЮрЃњрЃўрЃЎрЃљ|
|**Security / Permissions**|рЃњрЃљрЃюрЃљрЃАрЃ«рЃЋрЃљрЃЋрЃћрЃЉрЃА root-рЃАрЃљ рЃЊрЃљ user-рЃА, рЃўрЃфрЃљрЃЋрЃА рЃърЃарЃЮрЃфрЃћрЃАрЃћрЃЉрЃА рЃћрЃарЃЌрЃЏрЃљрЃюрЃћрЃЌрЃўрЃАрЃњрЃљрЃю|
|**System Calls API**|рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃћрЃЉрЃў Kernel-рЃАрЃЌрЃљрЃю рЃЎрЃЮрЃЏрЃБрЃюрЃўрЃЎрЃљрЃфрЃўрЃљрЃА рЃљрЃ«рЃЮрЃарЃфрЃўрЃћрЃџрЃћрЃЉрЃћрЃю рЃљрЃЏ рЃњрЃќрЃўрЃЌ (syscalls)|

## ­ЪњА рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:

рЃарЃЮрЃфрЃљ рЃерЃћрЃюрЃў рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљ (`myapp.exe`) рЃгрЃћрЃарЃА рЃцрЃљрЃўрЃџрЃА рЃЊрЃўрЃАрЃЎрЃќрЃћ:

1. рЃерЃћрЃюрЃў рЃЎрЃЮрЃЊрЃў рЃўрЃФрЃљрЃ«рЃћрЃЉрЃА `.WriteFile()` рЃцрЃБрЃюрЃЦрЃфрЃўрЃљрЃА (User Space)
    
2. рЃћрЃА рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ рЃЏрЃўрЃЊрЃўрЃА Kernel-рЃерЃў рЃарЃЮрЃњрЃЮрЃарЃф `sys_write()` system call
    
3. Kernel рЃљрЃњрЃќрЃљрЃЋрЃюрЃўрЃА рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљрЃА рЃЊрЃўрЃАрЃЎрЃўрЃА рЃЊрЃарЃљрЃўрЃЋрЃћрЃарЃќрЃћ
    
4. рЃЊрЃарЃљрЃўрЃЋрЃћрЃарЃў рЃБрЃЎрЃљрЃЋрЃерЃўрЃарЃЊрЃћрЃЉрЃљ рЃцрЃўрЃќрЃўрЃЎрЃБрЃа рЃЊрЃўрЃАрЃЎрЃА рЃЊрЃљ рЃгрЃћрЃарЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃА
    
5. Kernel рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА рЃърЃљрЃАрЃБрЃ«рЃА рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљрЃерЃў Рђћ РђюрЃгрЃљрЃарЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌ!РђЮ
    

> рЃљрЃюрЃБ рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљ **рЃљрЃарЃљрЃАрЃЊрЃарЃЮрЃА рЃћрЃ«рЃћрЃЉрЃљ рЃърЃўрЃарЃЊрЃљрЃърЃўрЃа рЃ░рЃљрЃарЃЊрЃЋрЃћрЃарЃА** Рђћ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ Kernel-рЃА рЃћрЃАрЃљрЃБрЃЉрЃарЃћрЃЉрЃљ.

## ­Ъћ╣ рЃЎрЃћрЃарЃюрЃћрЃџрЃўрЃА рЃбрЃўрЃърЃћрЃЉрЃў (рЃњрЃарЃљрЃюрЃБрЃџрЃљрЃарЃБрЃџрЃў рЃЊрЃЮрЃюрЃўрЃАрЃЌрЃЋрЃўрЃА)

|рЃбрЃўрЃърЃў|рЃљрЃ«рЃАрЃюрЃљ|рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃћрЃЉрЃў|
|---|---|---|
|**Monolithic Kernel**|рЃДрЃЋрЃћрЃџрЃљрЃцрЃћрЃарЃў рЃћрЃарЃЌ рЃЊрЃўрЃЊ рЃЉрЃўрЃарЃЌрЃЋрЃерЃўрЃљ (рЃЊрЃарЃљрЃўрЃЋрЃћрЃарЃћрЃЉрЃўрЃф рЃерЃўрЃњрЃюрЃўрЃЌ)|Linux, BSD|
|**Microkernel**|рЃЏрЃўрЃюрЃўрЃЏрЃљрЃџрЃБрЃарЃў рЃЉрЃўрЃарЃЌрЃЋрЃў Рђћ рЃЊрЃљрЃюрЃљрЃарЃЕрЃћрЃюрЃў рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃљрЃЊ|Minix, QNX|
|**Hybrid Kernel**|рЃерЃБрЃљрЃџрЃћрЃЊрЃБрЃарЃў рЃЋрЃљрЃарЃўрЃљрЃюрЃбрЃў|Windows NT, macOS XNU|

## ­Ъћ╣ Linux-рЃерЃў рЃарЃЮрЃњрЃЮрЃа рЃЕрЃљрЃюрЃА рЃЎрЃћрЃарЃюрЃћрЃџрЃў

- рЃбрЃћрЃарЃЏрЃўрЃюрЃљрЃџрЃерЃў `uname -r` рЃњрЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃА рЃЎрЃћрЃарЃюрЃћрЃџрЃўрЃА рЃЋрЃћрЃарЃАрЃўрЃљрЃА
    
    ```
    6.8.0-31-generic
    ```
    
- рЃћрЃА рЃюрЃўрЃерЃюрЃљрЃЋрЃА:
    
    - 6.8 = major рЃЋрЃћрЃарЃАрЃўрЃљ
        
    - РђюgenericРђЮ = рЃЏрЃЮрЃЊрЃБрЃџрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃљрЃА рЃБрЃГрЃћрЃарЃА рЃЏрЃ«рЃљрЃарЃА
        

Linux-рЃерЃў рЃЎрЃћрЃарЃюрЃћрЃџрЃў = рЃцрЃљрЃўрЃџрЃў `/boot/vmlinuz-*`  
рЃљрЃЏ рЃцрЃљрЃўрЃџрЃўрЃЌ рЃўрЃгрЃДрЃћрЃЉрЃљ рЃЉрЃБрЃЌрЃўрЃАрЃљрЃА рЃДрЃЋрЃћрЃџрЃљ рЃърЃарЃЮрЃфрЃћрЃАрЃў.

## ­Ъћ╣ рЃарЃљрЃбрЃЮрЃЏ рЃљрЃарЃўрЃА рЃћрЃА Docker/VM рЃЎрЃЮрЃюрЃбрЃћрЃЦрЃАрЃбрЃерЃў рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃў

- **VM-рЃА рЃљрЃЦрЃЋрЃА рЃЌрЃљрЃЋрЃўрЃАрЃў рЃЎрЃћрЃарЃюрЃћрЃџрЃў.**  
    рЃљрЃюрЃБ рЃЌрЃўрЃЌрЃЮ VM рЃЌрЃљрЃЋрЃўрЃАрЃў РђюрЃњрЃБрЃџрЃўРђЮ Рђћ рЃЊрЃљрЃЏрЃЮрЃБрЃЎрЃўрЃЊрЃћрЃЉрЃћрЃџрЃў рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃБрЃџрЃў рЃАрЃўрЃАрЃбрЃћрЃЏрЃљрЃљ.
    
- **Docker рЃЎрЃЮрЃюрЃбрЃћрЃўрЃюрЃћрЃарЃћрЃЉрЃў рЃЎрЃў рЃњрЃљрЃќрЃўрЃљрЃарЃћрЃЉрЃБрЃџ рЃЎрЃћрЃарЃюрЃћрЃџрЃќрЃћ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃћрЃю.**  
    рЃљрЃЏрЃўрЃбрЃЮрЃЏ Docker рЃЉрЃћрЃЋрЃарЃљрЃЊ рЃЏрЃАрЃБрЃЉрЃБрЃЦрЃўрЃљ Рђћ рЃљрЃа рЃАрЃГрЃўрЃарЃЊрЃћрЃЉрЃљ рЃЌрЃљрЃЋрЃўрЃАрЃў рЃЎрЃћрЃарЃюрЃћрЃџрЃў, рЃўрЃДрЃћрЃюрЃћрЃЉрЃА рЃ░рЃЮрЃАрЃб OS-рЃўрЃАрЃљрЃА.
