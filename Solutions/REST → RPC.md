рЃФрЃљрЃџрЃўрЃљрЃю рЃќрЃБрЃАрЃбрЃљрЃЊ рЃњрЃљрЃўрЃњрЃћ ­Ъј»  
рЃерЃћрЃю рЃБрЃЎрЃЋрЃћ рЃЊрЃљрЃўрЃГрЃўрЃарЃћ рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃљрЃќрЃарЃў Рђћ  
REST рЃќрЃБрЃАрЃбрЃљрЃЊ **РђюрЃњрЃљрЃарЃћрЃЊрЃљрЃю рЃерЃћрЃАрЃљрЃАрЃЋрЃџрЃћрЃџрЃў рЃЎрЃљрЃарЃўрЃљРђЮ**,  
рЃ«рЃЮрЃџрЃЮ RPC рЃљрЃарЃўрЃА **РђюрЃАрЃћрЃарЃЋрЃўрЃАрЃћрЃЉрЃА рЃерЃЮрЃарЃўрЃА рЃерЃўрЃЊрЃљ рЃАрЃљрЃБрЃЉрЃљрЃарЃўРђЮ**.

рЃљрЃ«рЃџрЃљ рЃљрЃњрЃўрЃ«рЃАрЃюрЃў рЃарЃљрЃбрЃЮрЃЏ рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃћрЃю рЃљрЃЏрЃљрЃА Рђћ рЃљрЃюрЃБ рЃарЃљрЃбрЃЮрЃЏ рЃљрЃЏрЃљрЃбрЃћрЃЉрЃћрЃю рЃљрЃЏ **рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌ рЃ░рЃЮрЃърЃА (hop)**, рЃЌрЃўрЃЌрЃЦрЃЮрЃА рЃќрЃћрЃЊрЃЏрЃћрЃбрЃў рЃцрЃћрЃюрЃљ рЃўрЃДрЃЮрЃА.  
рЃарЃћрЃљрЃџрЃБрЃарЃљрЃЊ рЃћрЃА рЃљрЃарЃўрЃА рЃљрЃБрЃфрЃўрЃџрЃћрЃЉрЃћрЃџрЃў рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃБрЃџрЃў рЃњрЃљрЃЊрЃљрЃгрЃДрЃЋрЃћрЃбрЃўрЃџрЃћрЃЉрЃљ рЃЊрЃўрЃЊрЃў рЃАрЃўрЃАрЃбрЃћрЃЏрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА.

## РџЎ№ИЈ рЃарЃљрЃбрЃЮрЃЏ рЃўрЃДрЃћрЃюрЃћрЃЉрЃћрЃю REST + RPC рЃћрЃарЃЌрЃљрЃЊ (рЃарЃљрЃбрЃЮрЃЏ рЃљрЃарЃўрЃА рЃћрЃА Рђюdouble hopРђЮ рЃљрЃБрЃфрЃўрЃџрЃћрЃЉрЃћрЃџрЃў)

|рЃЏрЃўрЃќрЃћрЃќрЃў|рЃљрЃ«рЃАрЃюрЃљ|
|---|---|
|­ЪДЕ **рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃЮрЃцрЃљ (Decoupling)**|REST API рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃћрЃЉрЃЌрЃљрЃю рЃЎрЃЮрЃЏрЃБрЃюрЃўрЃЎрЃљрЃфрЃўрЃљрЃќрЃћрЃљ рЃърЃљрЃАрЃБрЃ«рЃўрЃАрЃЏрЃњрЃћрЃЉрЃћрЃџрЃў, рЃ«рЃЮрЃџрЃЮ рЃЉрЃўрЃќрЃюрЃћрЃА рЃџрЃЮрЃњрЃўрЃЎрЃљ (Order, Payment, User...) рЃфрЃљрЃџрЃЎрЃћ рЃАрЃћрЃарЃЋрЃўрЃАрЃћрЃЉрЃерЃўрЃљ рЃњрЃљрЃДрЃЮрЃцрЃўрЃџрЃў. рЃћрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃўрЃФрЃџрЃћрЃЋрЃљ рЃњрЃБрЃюрЃЊрЃћрЃЉрЃў рЃЊрЃљрЃЏрЃЮрЃБрЃЎрЃўрЃЊрЃћрЃЉрЃџрЃљрЃЊ рЃўрЃЏрЃБрЃерЃљрЃЮрЃю рЃЊрЃљ рЃњрЃљрЃюрЃљрЃљрЃ«рЃџрЃЮрЃю рЃЌрЃљрЃЋрЃўрЃљрЃюрЃЌрЃў рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃў.|
|РџЎ№ИЈ **рЃњрЃљрЃЊрЃљрЃюрЃљрЃгрЃўрЃџрЃћрЃЉрЃБрЃџрЃў рЃЊрЃљрЃбрЃЋрЃўрЃарЃЌрЃЋрЃљ (Scalability)**|REST Gateway рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ РђюрЃАрЃљрЃцрЃљрЃарЃўРђЮрЃљ. рЃарЃЮрЃфрЃљ рЃљрЃЌрЃљрЃАрЃЮрЃЉрЃўрЃЌ рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃљ рЃЏрЃЮрЃЊрЃўрЃА, рЃЌрЃўрЃЌрЃЮ рЃАрЃћрЃарЃЋрЃўрЃАрЃў рЃЊрЃљрЃЏрЃЮрЃБрЃЎрЃўрЃЊрЃћрЃЉрЃџрЃљрЃЊ рЃўрЃќрЃарЃЊрЃћрЃЉрЃљ (рЃЏрЃљрЃњ. OrderService scale-out RPC-рЃўрЃА рЃЊрЃЮрЃюрЃћрЃќрЃћ).|
|­ЪДа **Single Responsibility (Clean Architecture)**|Gateway рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ РђюрЃЏрЃўрЃЏрЃљрЃарЃЌрЃЋрЃљрЃАРђЮ рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃА, рЃљрЃа рЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃА рЃџрЃЮрЃњрЃўрЃЎрЃљрЃА. рЃЌрЃЋрЃўрЃЌрЃЮрЃю рЃЉрЃўрЃќрЃюрЃћрЃА рЃџрЃЮрЃњрЃўрЃЎрЃљ рЃбрЃарЃўрЃљрЃџрЃћрЃЉрЃА рЃерЃўрЃЊрЃљ рЃАрЃћрЃарЃЋрЃўрЃАрЃћрЃЉрЃерЃў, рЃАрЃљрЃЊрЃљрЃф рЃБрЃЎрЃћрЃЌ рЃЎрЃЮрЃюрЃбрЃарЃЮрЃџрЃЊрЃћрЃЉрЃљ рЃбрЃарЃљрЃюрЃќрЃљрЃЦрЃфрЃўрЃћрЃЉрЃў, рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў, рЃЊрЃљ Domain Events.|
|­ЪД▒ **Reusable logic (Microservices)**|рЃАрЃ«рЃЋрЃљ рЃАрЃћрЃарЃЋрЃўрЃАрЃЏрЃљрЃф рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃўрЃАрЃљрЃарЃњрЃћрЃЉрЃџрЃЮрЃА рЃўрЃЏрЃљрЃЋрЃћ рЃџрЃЮрЃњрЃўрЃЎрЃўрЃЌ RPC-рЃўрЃА рЃЏрЃћрЃерЃЋрЃћрЃЮрЃЉрЃўрЃЌ (рЃЏрЃљрЃњ. Billing Рєњ User Рєњ Notification). REST рЃљрЃЦ рЃљрЃарЃљрЃцрЃћрЃарЃА рЃерЃБрЃљрЃерЃўрЃљ.|
|­ЪД« **Performance / Serialization**|REST Рєњ JSON Рєњ рЃюрЃћрЃџрЃў, рЃБрЃцрЃарЃЮ рЃЏрЃФрЃўрЃЏрЃћ.|

|рЃљрЃюрЃБ рЃерЃћрЃюрЃў РђюрЃЉрЃўрЃќрЃюрЃћрЃА рЃбрЃЋрЃўрЃюрЃўРђЮ рЃњрЃљрЃарЃћрЃЊрЃљрЃю рЃЊрЃљрЃфрЃБрЃџрЃўрЃљ.||

## ­Ъј» рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:

     РєЊ
 [ REST API Gateway ]   РєљРђћ>   [ AuthService ]
          РєЊ                        [ OrderService ]
          РєЊ                        [ UserService ]
      (Public)                    (Internal RPC)

­ЪЉЅ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃў REST API-рЃА рЃЏрЃўрЃЏрЃљрЃарЃЌрЃљрЃЋрЃА  
­ЪЉЅ REST API рЃерЃўрЃЊрЃљ рЃЦрЃАрЃћрЃџрЃерЃў рЃљрЃњрЃќрЃљрЃЋрЃюрЃўрЃА RPC call-рЃА рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџ рЃАрЃћрЃарЃЋрЃўрЃАрЃќрЃћ  
­ЪЉЅ рЃерЃћрЃЊрЃћрЃњрЃў рЃБрЃЉрЃарЃБрЃюрЃЊрЃћрЃЉрЃљ REST-рЃА рЃЊрЃљ рЃўрЃЦрЃўрЃЊрЃљрЃю Рђћ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃА JSON рЃцрЃЮрЃарЃЏрЃљрЃбрЃерЃў

## ­Ъџђ рЃарЃљрЃбрЃЮрЃЏ рЃ»рЃЮрЃЉрЃА рЃљрЃАрЃћ:

1. **рЃерЃћрЃю рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљ рЃАрЃћрЃарЃЋрЃўрЃАрЃћрЃЉрЃў рЃЊрЃљрЃЏрЃЮрЃБрЃЎрЃўрЃЊрЃћрЃЉрЃџрЃљрЃЊ рЃњрЃљрЃюрЃљрЃљрЃ«рЃџрЃЮ** Рђћ Gateway рЃўрЃфрЃўрЃА рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃЎрЃЮрЃюрЃбрЃарЃљрЃЦрЃбрЃў.
    
2. **рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃЊрЃљрЃљрЃЏрЃљрЃбрЃЮ рЃљрЃ«рЃљрЃџрЃў REST endpoint-рЃћрЃЉрЃў RPC-рЃўрЃА рЃерЃћрЃфрЃЋрЃџрЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ.**
    
3. **рЃерЃўрЃЊрЃљ рЃАрЃћрЃарЃЋрЃўрЃАрЃћрЃЉрЃў рЃЎрЃЮрЃЏрЃБрЃюрЃўрЃЎрЃљрЃфрЃўрЃљрЃА рЃБрЃцрЃарЃЮ рЃАрЃгрЃарЃљрЃцрЃљрЃЊ рЃљрЃ«рЃЮрЃарЃфрЃўрЃћрЃџрЃћрЃЉрЃћрЃю.**
    
4. **рЃерЃћрЃюрЃўрЃдрЃЉрЃБрЃџрЃўрЃљ рЃерЃўрЃЊрЃљ рЃўрЃюрЃцрЃарЃљрЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ Рђћ рЃњрЃљрЃарЃћрЃЊрЃљрЃю рЃЋрЃћрЃарЃљрЃЋрЃўрЃю рЃерЃћрЃћрЃ«рЃћрЃЉрЃљ OrderService-рЃА рЃърЃўрЃарЃЊрЃљрЃърЃўрЃа.**

## ­ЪЊд рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў рЃърЃарЃљрЃЦрЃбрЃўрЃЎрЃљрЃерЃў (рЃњрЃљрЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃБрЃџрЃў рЃюрЃљрЃЎрЃљрЃЊрЃў)

```csharp
// REST Gateway Controller
[HttpPost("orders")]
public async Task<IActionResult> CreateOrder(CreateOrderDto dto)
{
    var result = await _orderGrpc.CreateOrderAsync(new CreateOrderRequest {
        UserId = dto.UserId,
        ProductId = dto.ProductId
    });
    return Ok(result);
}
```

РъА рЃљрЃЦ REST рЃўрЃдрЃћрЃЉрЃА HTTP рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃљрЃА  
РъА Gateway рЃљрЃњрЃќрЃљрЃЋрЃюрЃўрЃА **RPC call-рЃА** рЃерЃўрЃЊрЃљ OrderService-рЃќрЃћ  
РъА OrderService рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃА рЃарЃћрЃљрЃџрЃБрЃа рЃЉрЃўрЃќрЃюрЃћрЃА рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃљрЃА (DB, Payment, etc.)
