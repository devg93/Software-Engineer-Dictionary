рЃФрЃљрЃџрЃўрЃљрЃю рЃЎрЃљрЃарЃњрЃў Рђћ ­ЪЉЈ рЃљрЃ«рЃџрЃљ рЃЋрЃўрЃџрЃљрЃърЃљрЃарЃљрЃЎрЃЮрЃЌ рЃћрЃарЃЌ рЃФрЃљрЃџрЃўрЃљрЃю рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃю рЃбрЃћрЃарЃЏрЃўрЃюрЃќрЃћ:

---

## ­ЪњА **рЃЎрЃЮрЃ░рЃћрЃќрЃўрЃљ (Cohesion)**

**рЃЏрЃЮрЃЎрЃџрЃћ рЃњрЃљрЃюрЃЏрЃљрЃарЃбрЃћрЃЉрЃљ:**  
­ЪЉЅ _рЃЎрЃЮрЃ░рЃћрЃќрЃўрЃљ_ рЃюрЃўрЃерЃюрЃљрЃЋрЃА **рЃарЃљрЃЏрЃЊрЃћрЃюрЃљрЃЊ рЃЎрЃљрЃарЃњрЃљрЃЊ рЃБрЃЎрЃљрЃЋрЃерЃўрЃарЃЊрЃћрЃЉрЃљ рЃћрЃарЃЌрЃў рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃА рЃерЃўрЃњрЃюрЃўрЃЌ рЃљрЃарЃАрЃћрЃЉрЃБрЃџрЃў рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃў рЃћрЃарЃЌрЃЏрЃљрЃюрЃћрЃЌрЃА.**

рЃљрЃюрЃБ Рђћ рЃарЃљрЃЏрЃЊрЃћрЃюрЃљрЃЊ "рЃћрЃарЃЌ рЃЏрЃўрЃќрЃљрЃюрЃА" рЃћрЃЏрЃАрЃљрЃ«рЃБрЃарЃћрЃЉрЃљ рЃЎрЃџрЃљрЃАрЃўрЃА рЃљрЃю рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃўрЃА рЃерЃўрЃњрЃюрЃўрЃЌ рЃљрЃарЃАрЃћрЃЉрЃБрЃџрЃў рЃЏрЃћрЃЌрЃЮрЃЊрЃћрЃЉрЃў рЃЊрЃљ рЃЋрЃћрЃџрЃћрЃЉрЃў.

---

### ­Ъј» **рЃўрЃЊрЃћрЃљ рЃЏрЃљрЃарЃбрЃўрЃЋрЃљрЃЊ**

рЃЌрЃБ рЃЎрЃџрЃљрЃАрЃерЃў рЃљрЃю рЃЏрЃЮрЃЊрЃБрЃџрЃерЃў рЃДрЃЋрЃћрЃџрЃљ рЃЏрЃћрЃЌрЃЮрЃЊрЃў рЃЊрЃљ рЃЋрЃћрЃџрЃў рЃћрЃарЃЌрЃАрЃљ рЃЊрЃљ рЃўрЃЏрЃљрЃЋрЃћ рЃЏрЃўрЃќрЃљрЃюрЃА рЃћрЃЏрЃАрЃљрЃ«рЃБрЃарЃћрЃЉрЃљ Рђћ  
рЃЏрЃљрЃдрЃљрЃџрЃў рЃЎрЃЮрЃ░рЃћрЃќрЃўрЃљ рЃњрЃљрЃЦрЃЋрЃА РюЁ  
рЃЌрЃБ рЃЎрЃў рЃўрЃАрЃўрЃюрЃў рЃћрЃарЃЌрЃЏрЃљрЃюрЃћрЃЌрЃЌрЃљрЃю рЃЎрЃљрЃЋрЃерЃўрЃарЃерЃў рЃљрЃа рЃљрЃарЃўрЃљрЃю, рЃЊрЃљ рЃБрЃЉрЃарЃљрЃџрЃЮрЃЊ "рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃўрЃЌ" рЃћрЃарЃЌ рЃцрЃљрЃўрЃџрЃерЃў рЃфрЃ«рЃЮрЃЋрЃарЃЮрЃЉрЃћрЃю Рђћ  
рЃЊрЃљрЃЉрЃљрЃџрЃў рЃЎрЃЮрЃ░рЃћрЃќрЃўрЃљрЃљ РЮї

---

### ­ЪЊд **рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў рЃЊрЃљрЃЉрЃљрЃџрЃў рЃЎрЃЮрЃ░рЃћрЃќрЃўрЃўрЃА**

```csharp
public class Utility
{
    public void SendEmail(string to, string body) { ... }
    public void CalculateTax(decimal price) { ... }
    public void LogError(string message) { ... }
}
```

РъА рЃљрЃЦ рЃЏрЃћрЃЌрЃЮрЃЊрЃћрЃЉрЃА рЃерЃЮрЃарЃўрЃА рЃљрЃарЃљрЃюрЃљрЃўрЃарЃў рЃЎрЃљрЃЋрЃерЃўрЃарЃў рЃљрЃа рЃљрЃарЃўрЃА Рђћ  
рЃћрЃА рЃЎрЃџрЃљрЃАрЃўрЃА рЃЏрЃўрЃќрЃљрЃюрЃў рЃњрЃљрЃБрЃарЃЎрЃЋрЃћрЃЋрЃћрЃџрЃўрЃљ.  
рЃћрЃА рЃљрЃарЃўрЃА **рЃЊрЃљрЃЉрЃљрЃџрЃў рЃЎрЃЮрЃ░рЃћрЃќрЃўрЃљ** Рђћ God Class рЃАрЃбрЃўрЃџрЃўрЃА рЃюрЃўрЃерЃљрЃюрЃў.

---

### ­ЪДЕ **рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў рЃЏрЃљрЃдрЃљрЃџрЃў рЃЎрЃЮрЃ░рЃћрЃќрЃўрЃўрЃА**

```csharp
public class EmailService
{
    public void SendEmail(string to, string body) { ... }
    public void ValidateEmail(string email) { ... }
    public Task<bool> SendBulkEmailsAsync(List<string> recipients) { ... }
}
```

РъА рЃДрЃЋрЃћрЃџрЃљ рЃЏрЃћрЃЌрЃЮрЃЊрЃў рЃћрЃарЃЌ рЃЏрЃўрЃќрЃљрЃюрЃА рЃћрЃЏрЃАрЃљрЃ«рЃБрЃарЃћрЃЉрЃљ Рђћ _email рЃњрЃљрЃњрЃќрЃљрЃЋрЃюрЃљрЃА._  
рЃћрЃА рЃљрЃарЃўрЃА **рЃЏрЃљрЃдрЃљрЃџрЃў рЃЎрЃЮрЃ░рЃћрЃќрЃўрЃљ**, рЃљрЃюрЃБ рЃАрЃБрЃцрЃЌрЃљ рЃЊрЃљ рЃљрЃЊрЃЋрЃўрЃџрЃљрЃЊ рЃњрЃљрЃАрЃљрЃњрЃћрЃЉрЃў рЃЊрЃўрЃќрЃљрЃўрЃюрЃў.

---

### Рџќ№ИЈ **рЃЎрЃЮрЃ░рЃћрЃќрЃўрЃљ vs рЃЎрЃБрЃърЃџрЃўрЃюрЃњрЃў**

|рЃбрЃћрЃарЃЏрЃўрЃюрЃў|рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ|
|---|---|
|**Cohesion (рЃЎрЃЮрЃ░рЃћрЃќрЃўрЃљ)**|рЃарЃљрЃЏрЃЊрЃћрЃюрЃљрЃЊ рЃЎрЃљрЃарЃњрЃљрЃЊрЃљрЃљ рЃЊрЃљрЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃЎрЃџрЃљрЃАрЃўрЃА рЃерЃўрЃњрЃюрЃўрЃЌ рЃљрЃарЃАрЃћрЃЉрЃБрЃџрЃў рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃў рЃћрЃарЃЌрЃЏрЃљрЃюрЃћрЃЌрЃЌрЃљрЃю|
|**Coupling (рЃЎрЃБрЃърЃџрЃўрЃюрЃњрЃў)**|рЃарЃљрЃЏрЃЊрЃћрЃюрЃљрЃЊ рЃФрЃџрЃўрЃћрЃарЃљрЃЊ рЃљрЃарЃўрЃА рЃЊрЃљрЃЏрЃЮрЃЎрЃўрЃЊрЃћрЃЉрЃБрЃџрЃў рЃћрЃарЃЌрЃў рЃЎрЃџрЃљрЃАрЃў/рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃЏрЃћрЃЮрЃарЃћрЃќрЃћ|

­ЪЉЅ рЃЏрЃўрЃќрЃљрЃюрЃў рЃДрЃЮрЃЋрЃћрЃџрЃЌрЃЋрЃўрЃА рЃљрЃАрЃћрЃЌрЃўрЃљ:

> **High Cohesion, Low Coupling**  
> Рђћ рЃљрЃюрЃБ рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃўрЃДрЃЮрЃА рЃерЃўрЃњрЃюрЃўрЃЊрЃљрЃю рЃФрЃџрЃўрЃћрЃарЃљрЃЊ рЃерЃћрЃЎрЃарЃБрЃџрЃў, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃњрЃљрЃарЃћрЃЊрЃљрЃю рЃЊрЃљрЃЏрЃЮрЃБрЃЎрЃўрЃЊрЃћрЃЉрЃћрЃџрЃў.

---

### ­ЪЈЌ№ИЈ **рЃърЃарЃљрЃЦрЃбрЃўрЃЎрЃБрЃџрЃў рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў (Clean Architecture)**

рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ, рЃерЃћрЃюрЃў `UserValidationBackgroundService` рЃљрЃарЃўрЃА **рЃЏрЃљрЃдрЃљрЃџрЃў рЃЎрЃЮрЃ░рЃћрЃќрЃўрЃўрЃА рЃЏрЃЦрЃЮрЃюрЃћ**:

- рЃДрЃЋрЃћрЃџрЃљ private рЃЏрЃћрЃЌрЃЮрЃЊрЃў (`ProcessOutboxEvents`, `TrySaveChangesAsync`, `TriggerRecoveryAsync`) рЃћрЃЏрЃАрЃљрЃ«рЃБрЃарЃћрЃЉрЃљ рЃћрЃарЃЌ рЃЏрЃўрЃќрЃљрЃюрЃА Рђћ рЃўрЃБрЃќрЃћрЃарЃўрЃА рЃўрЃЋрЃћрЃюрЃЌрЃћрЃЉрЃўрЃА рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃљрЃА.
    
- рЃўрЃА рЃљрЃа рЃБрЃарЃћрЃЋрЃА рЃАрЃ«рЃЋрЃљ рЃЉрЃўрЃќрЃюрЃћрЃА рЃџрЃЮрЃњрЃўрЃЎрЃљрЃА (рЃЏрЃљрЃњ. Auth рЃљрЃю DeviceControl).
    

рЃћрЃА рЃюрЃўрЃерЃюрЃљрЃЋрЃА, рЃарЃЮрЃЏ рЃћрЃА рЃАрЃћрЃарЃЋрЃўрЃАрЃў:

- рЃљрЃЊрЃЋрЃўрЃџрЃљрЃЊ рЃўрЃгрЃћрЃарЃћрЃЉрЃљ,
    
- рЃљрЃЊрЃЋрЃўрЃџрЃљрЃЊ рЃўрЃерЃџрЃћрЃЉрЃљ рЃбрЃћрЃАрЃбрЃћрЃЉрЃљрЃЊ,
    
- рЃЏрЃљрЃарЃбрЃўрЃЋрЃљрЃЊ рЃўрЃерЃџрЃћрЃЉрЃљ рЃАрЃ«рЃЋрЃљ рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃљрЃЊ.
    

---

### ­ЪДа **рЃерЃћрЃ»рЃљрЃЏрЃћрЃЉрЃљ**

|рЃЌрЃЋрЃўрЃАрЃћрЃЉрЃљ|рЃЏрЃљрЃдрЃљрЃџрЃў рЃЎрЃЮрЃ░рЃћрЃќрЃўрЃљ|рЃЊрЃљрЃЉрЃљрЃџрЃў рЃЎрЃЮрЃ░рЃћрЃќрЃўрЃљ|
|---|---|---|
|рЃЏрЃўрЃќрЃљрЃюрЃў|рЃћрЃарЃЌрЃў рЃЊрЃљ рЃЏрЃЎрЃљрЃцрЃўрЃЮ|рЃљрЃарЃљрЃћрЃарЃЌрЃњрЃЋрЃљрЃарЃЮрЃЋрЃљрЃюрЃў|
|рЃгрЃЋрЃЊрЃЮрЃЏрЃљ|рЃерЃўрЃЊрЃљ рЃЊрЃљрЃЏрЃЮрЃЎрЃўрЃЊрЃћрЃЉрЃБрЃџрЃћрЃЉрЃћрЃЉрЃў|рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃўрЃЌрЃў рЃБрЃарЃЌрЃўрЃћрЃарЃЌрЃЮрЃЉрЃћрЃЉрЃў|
|рЃЎрЃЮрЃЊрЃўрЃА рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃљ|рЃЏрЃљрЃарЃбрЃўрЃЋрЃў|рЃарЃЌрЃБрЃџрЃў|
|рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃўрЃА рЃљрЃџрЃЉрЃљрЃЌрЃЮрЃЉрЃљ|рЃюрЃљрЃЎрЃџрЃћрЃЉрЃў|рЃЏрЃћрЃбрЃў|
|рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃљ|рЃАрЃБрЃцрЃЌрЃљ рЃЊрЃљ рЃЏрЃЮрЃЊрЃБрЃџрЃБрЃарЃў|God Class / spaghetti code|
