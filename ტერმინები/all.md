## ­ЪДа **рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃбрЃћрЃарЃЏрЃўрЃюрЃћрЃЉрЃў (Core Concepts)**

|рЃбрЃћрЃарЃЏрЃўрЃюрЃў|рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ|
|---|---|
|**Thread**|рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃА рЃерЃћрЃАрЃарЃБрЃџрЃћрЃЉрЃўрЃА рЃЏрЃфрЃўрЃарЃћ рЃЊрЃљрЃЏрЃЮрЃБрЃЎрЃўрЃЊрЃћрЃЉрЃћрЃџрЃў рЃюрЃљрЃЎрЃљрЃЊрЃў. рЃЏрЃарЃљрЃЋрЃљрЃџрЃбрЃарЃћрЃЊрЃўрЃљрЃюрЃЮрЃЉрЃљ рЃќрЃарЃЊрЃўрЃА рЃАрЃўрЃЕрЃЦрЃљрЃарЃћрЃА, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃарЃЌрЃБрЃџрЃЊрЃћрЃЉрЃљ рЃАрЃўрЃюрЃЦрЃарЃЮрЃюрЃўрЃќрЃљрЃфрЃўрЃљ.|
|**Process**|рЃЊрЃљрЃЏрЃЮрЃБрЃЎрЃўрЃЊрЃћрЃЉрЃћрЃџрЃў рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ рЃЌрЃљрЃЋрЃўрЃАрЃў рЃЏрЃћрЃ«рЃАрЃўрЃћрЃарЃћрЃЉрЃўрЃЌрЃљ рЃЊрЃљ рЃарЃћрЃАрЃБрЃарЃАрЃћрЃЉрЃўрЃЌ.|
|**Race Condition**|рЃарЃЮрЃфрЃљ рЃЮрЃарЃў рЃљрЃю рЃЏрЃћрЃбрЃў thread рЃћрЃарЃЌрЃЊрЃарЃЮрЃБрЃџрЃљрЃЊ рЃфрЃЋрЃџрЃўрЃА рЃћрЃарЃЌрЃАрЃљ рЃЊрЃљ рЃўрЃЏрЃљрЃЋрЃћ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃА, рЃарЃўрЃА рЃњрЃљрЃЏрЃЮрЃф рЃерЃћрЃЊрЃћрЃњрЃў рЃљрЃарЃљрЃърЃарЃЮрЃњрЃюрЃЮрЃќрЃўрЃарЃћрЃЉрЃљрЃЊрЃўрЃљ.|
|**Deadlock**|рЃАрЃўрЃбрЃБрЃљрЃфрЃўрЃљ, рЃарЃЮрЃфрЃљ рЃЮрЃарЃў рЃърЃарЃЮрЃфрЃћрЃАрЃў рЃћрЃџрЃЮрЃЊрЃћрЃЉрЃљ рЃћрЃарЃЌрЃЏрЃљрЃюрЃћрЃЌрЃўрЃА рЃЏрЃўрЃћрЃа рЃЊрЃљрЃЎрЃљрЃЋрЃћрЃЉрЃБрЃџ рЃарЃћрЃАрЃБрЃарЃАрЃА Рђћ рЃЊрЃљ рЃљрЃарЃфрЃћрЃарЃЌрЃў рЃљрЃдрЃљрЃа рЃљрЃњрЃарЃФрЃћрЃџрЃћрЃЉрЃА.|
|**Atomic Operation**|рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃАрЃарЃБрЃџрЃЊрЃћрЃЉрЃљ рЃЏрЃЌрЃџрЃўрЃљрЃюрЃљрЃЊ рЃљрЃю рЃАрЃљрЃћрЃарЃЌрЃЮрЃЊ рЃљрЃа рЃАрЃарЃБрЃџрЃЊрЃћрЃЉрЃљ (рЃерЃћрЃгрЃДрЃЊрЃћрЃЉрЃљ рЃерЃБрЃљрЃерЃў рЃњрЃљрЃарЃћрЃерЃћ).|
|**Idempotent Operation**|рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃљ, рЃарЃЮрЃЏрЃџрЃўрЃА рЃЏрЃарЃљрЃЋрЃљрЃџрЃ»рЃћрЃа рЃЕрЃљрЃбрЃљрЃарЃћрЃЉрЃљ рЃўрЃФрЃџрЃћрЃЋрЃљ рЃћрЃарЃЌрЃАрЃљ рЃЊрЃљ рЃўрЃЏрЃљрЃЋрЃћ рЃерЃћрЃЊрЃћрЃњрЃА (рЃЏрЃљрЃњ. `PUT` API).|
|**Namespace**|рЃџрЃЮрЃњрЃўрЃЎрЃБрЃарЃў рЃАрЃўрЃЋрЃарЃфрЃћ рЃАрЃљрЃ«рЃћрЃџрЃћрЃЉрЃўрЃА рЃЎрЃЮрЃюрЃцрЃџрЃўрЃЦрЃбрЃўрЃА рЃЌрЃљрЃЋрЃўрЃЊрЃљрЃю рЃљрЃАрЃљрЃфрЃўрЃџрЃћрЃЉрЃџрЃљрЃЊ (рЃЏрЃљрЃњ. `System.Collections`).|

## ­ЪДЕ **Clean Architecture / DDD рЃбрЃћрЃарЃЏрЃўрЃюрЃћрЃЉрЃў**

|рЃбрЃћрЃарЃЏрЃўрЃюрЃў|рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ|
|---|---|
|**Entity**|рЃЊрЃЮрЃЏрЃћрЃюрЃўрЃА рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃў, рЃарЃЮрЃЏрЃћрЃџрЃАрЃљрЃф рЃљрЃЦрЃЋрЃА рЃўрЃЊрЃћрЃюрЃбрЃўрЃцрЃўрЃЎрЃљрЃбрЃЮрЃарЃў (`Id`) рЃЊрЃљ рЃфрЃ«рЃЮрЃЋрЃарЃћрЃЉрЃўрЃА рЃфрЃўрЃЎрЃџрЃў (рЃЏрЃљрЃњ. `User`, `Order`).|
|**Value Object**|рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃў рЃўрЃЊрЃћрЃюрЃбрЃўрЃцрЃўрЃЎрЃљрЃбрЃЮрЃарЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ, рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃўрЃЌ рЃўрЃЊрЃћрЃюрЃбрЃБрЃарЃўрЃљ (рЃЏрЃљрЃњ. `Money`, `Address`).|
|**Aggregate Root**|рЃЊрЃЮрЃЏрЃћрЃюрЃўрЃА рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃЎрЃЋрЃљрЃюрЃФрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЏрЃљрЃарЃЌрЃљрЃЋрЃА рЃАрЃ«рЃЋрЃљ рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃћрЃЉрЃА рЃЌрЃљрЃЋрЃўрЃА рЃАрЃљрЃќрЃдрЃЋрЃарЃћрЃЉрЃерЃў.|
|**Repository**|рЃљрЃЉрЃАрЃбрЃарЃљрЃЦрЃфрЃўрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃцрЃљрЃарЃљрЃЋрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃљрЃАрЃЌрЃљрЃю рЃгрЃЋрЃЊрЃЮрЃЏрЃљрЃА (`Add`, `GetById`, `Save`).|
|**Unit of Work**|рЃбрЃарЃљрЃюрЃќрЃљрЃЦрЃфрЃўрЃўрЃА рЃЊрЃЮрЃюрЃћрЃќрЃћ рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃўрЃА рЃћрЃарЃЌрЃўрЃљрЃюрЃў рЃЏрЃљрЃарЃЌрЃЋрЃљ (`SaveChanges`).|
|**Domain Event**|рЃЏрЃЮрЃЋрЃџрЃћрЃюрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃгрЃљрЃарЃЏрЃЮрЃўрЃерЃЮрЃЉрЃљ рЃЊрЃЮрЃЏрЃћрЃюрЃерЃў (рЃЏрЃљрЃњ. `UserRegistered`).|
|**Outbox Pattern**|рЃўрЃЋрЃћрЃюрЃЌрЃћрЃЉрЃўрЃА рЃерЃћрЃюрЃљрЃ«рЃЋрЃљ рЃЉрЃљрЃќрЃљрЃерЃў, рЃарЃљрЃЌрЃљ рЃњрЃљрЃарЃљрЃюрЃбрЃўрЃарЃћрЃЉрЃБрЃџрЃљрЃЊ рЃњрЃљрЃўрЃњрЃќрЃљрЃЋрЃюрЃЮрЃА рЃАрЃ«рЃЋрЃљ рЃАрЃўрЃАрЃбрЃћрЃЏрЃћрЃЉрЃерЃў (рЃЏрЃљрЃерЃўрЃюрЃљрЃф рЃЎрЃў рЃЌрЃБ рЃАрЃћрЃарЃЋрЃўрЃАрЃў рЃЊрЃљрЃћрЃфрЃљ).|
|**Saga / Process Manager**|рЃњрЃарЃФрЃћрЃџрЃЋрЃљрЃЊрЃўрЃљрЃюрЃў рЃърЃарЃЮрЃфрЃћрЃАрЃўрЃА рЃЎрЃЮрЃюрЃбрЃарЃЮрЃџрЃў рЃЏрЃарЃљрЃЋрЃљрЃџрЃў рЃАрЃћрЃарЃЋрЃўрЃАрЃўрЃА/рЃбрЃарЃљрЃюрЃќрЃљрЃЦрЃфрЃўрЃўрЃА рЃњрЃљрЃюрЃЏрЃљрЃЋрЃџрЃЮрЃЉрЃљрЃерЃў.|

## РџЎ№ИЈ **Concurrency / Infrastructure рЃбрЃћрЃарЃЏрЃўрЃюрЃћрЃЉрЃў**

|рЃбрЃћрЃарЃЏрЃўрЃюрЃў|рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ|
|---|---|
|**Distributed Lock**|рЃЉрЃџрЃЮрЃЎрЃўрЃарЃћрЃЉрЃўрЃА рЃЏрЃћрЃЦрЃљрЃюрЃўрЃќрЃЏрЃў рЃЏрЃарЃљрЃЋрЃљрЃџ рЃАрЃћрЃарЃЋрЃћрЃарЃА рЃерЃЮрЃарЃўрЃА (рЃЏрЃљрЃњ. Redis).|
|**Leader Election**|рЃарЃЮрЃфрЃљ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃћрЃарЃЌрЃў рЃАрЃћрЃарЃЋрЃўрЃАрЃў рЃ«рЃЊрЃћрЃЉрЃљ рЃљрЃЦрЃбрЃўрЃБрЃарЃў рЃџрЃўрЃЊрЃћрЃарЃў рЃЊрЃўрЃАрЃбрЃарЃўрЃЉрЃБрЃфрЃўрЃБрЃџ рЃЎрЃџрЃљрЃАрЃбрЃћрЃарЃерЃў.|
|**Retry Policy**|рЃгрЃљрЃарЃБрЃЏрЃљрЃбрЃћрЃЉрЃћрЃџрЃў рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃўрЃА рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃў рЃ«рЃћрЃџрЃљрЃ«рЃљрЃџрЃў рЃфрЃЊрЃљ (рЃЏрЃљрЃњ. Polly).|
|**Circuit Breaker**|рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃА рЃЊрЃљрЃфрЃЋрЃўрЃА рЃЏрЃћрЃЦрЃљрЃюрЃўрЃќрЃЏрЃў Рђћ рЃЌрЃБ рЃЉрЃћрЃЋрЃарЃў рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљрЃљ, рЃЊрЃарЃЮрЃћрЃЉрЃўрЃЌ рЃљрЃЕрЃћрЃарЃћрЃЉрЃА рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃћрЃЉрЃА.|
|**Message Broker**|рЃљрЃАрЃўрЃюрЃЦрЃарЃЮрЃюрЃБрЃџрЃў рЃЎрЃЮрЃЏрЃБрЃюрЃўрЃЎрЃљрЃфрЃўрЃўрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ (RabbitMQ, Kafka).|
|**Eventual Consistency**|рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў рЃАрЃљрЃЉрЃЮрЃџрЃЮрЃЮрЃЊ рЃЊрЃљрЃАрЃўрЃюрЃЦрЃарЃЮрЃюрЃЊрЃћрЃЉрЃљ, рЃЌрЃБрЃЏрЃфрЃљ рЃљрЃарЃљ рЃЏрЃДрЃўрЃАрЃўрЃћрЃарЃљрЃЊ (рЃЏрЃљрЃњ. рЃЊрЃўрЃАрЃбрЃарЃўрЃЉрЃБрЃфрЃўрЃБрЃџ рЃАрЃўрЃАрЃбрЃћрЃЏрЃћрЃЉрЃерЃў).|

## ­Ъїљ **Web / API рЃбрЃћрЃарЃЏрЃўрЃюрЃћрЃЉрЃў**

|рЃбрЃћрЃарЃЏрЃўрЃюрЃў|рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ|
|---|---|
|**REST**|API рЃЊрЃўрЃќрЃљрЃўрЃюрЃўрЃА рЃАрЃбрЃўрЃџрЃў рЃарЃћрЃАрЃБрЃарЃАрЃћрЃЉрЃќрЃћ рЃЊрЃљрЃцрЃБрЃФрЃюрЃћрЃЉрЃўрЃЌ (`GET`, `POST`, `PUT`, `DELETE`).|
|**GraphQL**|рЃЏрЃЮрЃЦрЃюрЃўрЃџрЃў API рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃў, рЃАрЃљрЃЊрЃљрЃф рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃў рЃЌрЃљрЃЋрЃљрЃЊ рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃљрЃЋрЃА рЃарЃљ рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ рЃАрЃГрЃўрЃарЃЊрЃћрЃЉрЃљ.|
|**JWT (JSON Web Token)**|рЃљрЃЋрЃбрЃћрЃюрЃбрЃўрЃЎрЃљрЃфрЃўрЃўрЃА рЃбрЃЮрЃЎрЃћрЃюрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃерЃћрЃўрЃфрЃљрЃЋрЃА рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃўрЃЊрЃћрЃюрЃбрЃЮрЃЉрЃљрЃА.|
|**Rate Limiting**|рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃћрЃЉрЃўрЃА рЃарЃљрЃЮрЃЊрЃћрЃюрЃЮрЃЉрЃўрЃА рЃерЃћрЃќрЃдрЃБрЃЊрЃЋрЃљ рЃЊрЃарЃЮрЃўрЃА рЃћрЃарЃЌрЃћрЃБрЃџрЃерЃў (DDoS-рЃўрЃА рЃгрЃўрЃюрЃљрЃљрЃдрЃЏрЃЊрЃћрЃњ).|
|**Middleware**|Request/Response-рЃўрЃА рЃерЃБрЃљ рЃерЃарЃћ, рЃАрЃљрЃЊрЃљрЃф рЃ«рЃЊрЃћрЃЉрЃљ рЃџрЃЮрЃњрЃўрЃарЃћрЃЉрЃљ, рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃљ рЃЊрЃљ рЃАрЃ«рЃЋрЃљ.|

|**Singleton**|рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃў, рЃарЃЮрЃЏрЃџрЃўрЃА рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃћрЃарЃЌрЃў рЃћрЃњрЃќрЃћрЃЏрЃърЃџрЃљрЃарЃў рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃА рЃЏрЃЌрЃћрЃџ рЃљрЃърЃерЃў.|
|**Factory**|рЃЦрЃЏрЃюрЃўрЃА рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃћрЃЉрЃА рЃарЃЌрЃБрЃџрЃў рЃџрЃЮрЃњрЃўрЃЎрЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ.|
|**Observer**|рЃўрЃЋрЃћрЃюрЃЌрЃћрЃЉрЃќрЃћ рЃЊрЃљрЃДрЃарЃЊрЃюрЃЮрЃЉрЃўрЃџрЃў рЃЏрЃЮрЃЊрЃћрЃџрЃў (Publish/Subscribe).|
|**Strategy**|рЃљрЃџрЃњрЃЮрЃарЃўрЃЌрЃЏрЃўрЃА рЃЏрЃЮрЃЦрЃюрЃўрЃџрЃў рЃерЃћрЃфрЃЋрЃџрЃљ рЃерЃћрЃАрЃарЃБрЃџрЃћрЃЉрЃўрЃА рЃЊрЃарЃЮрЃА.|
|**Decorator**|рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃўрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃўрЃАрЃЌрЃЋрЃўрЃА рЃЏрЃћрЃЏрЃЎрЃЋрЃўрЃЊрЃарЃћрЃЮрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃерЃћрЃфрЃЋрЃџрЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ.|
|**Mediator / CQRS / Command Handler**|рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃћрЃЉрЃўрЃА/рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃћрЃЉрЃўрЃА рЃфрЃћрЃюрЃбрЃарЃљрЃџрЃўрЃќрЃћрЃЉрЃБрЃџрЃў рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃљ (рЃЏрЃљрЃњ. MediatR).|

## ­ЪД░ **Testing / CI/CD рЃбрЃћрЃарЃЏрЃўрЃюрЃћрЃЉрЃў**

|рЃбрЃћрЃарЃЏрЃўрЃюрЃў|рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ|
|---|---|
|**Unit Test**|рЃбрЃћрЃАрЃбрЃў рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџрЃў рЃЏрЃћрЃЌрЃЮрЃЊрЃўрЃА рЃљрЃю рЃЎрЃџрЃљрЃАрЃўрЃАрЃљрЃЌрЃЋрЃўрЃА.|
|**Integration Test**|рЃбрЃћрЃАрЃбрЃў рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃА рЃюрЃљрЃгрЃўрЃџрЃћрЃЉрЃА рЃћрЃарЃЌрЃљрЃЊ (рЃЏрЃљрЃњ. API + DB).|
|**End-to-End Test (E2E)**|рЃбрЃћрЃАрЃбрЃў рЃарЃћрЃљрЃџрЃБрЃарЃў рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃЦрЃфрЃћрЃЋрЃўрЃА рЃАрЃўрЃЏрЃБрЃџрЃљрЃфрЃўрЃўрЃЌ.|
|**Mock / Stub / Fake**|рЃцрЃўрЃЦрЃбрЃўрЃБрЃарЃў рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃћрЃЉрЃў рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА.|
|**Continuous Integration (CI)**|рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃў рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃљ рЃДрЃЮрЃЋрЃћрЃџ commit-рЃќрЃћ.|
|**Continuous Deployment (CD)**|рЃбрЃћрЃАрЃбрЃўрЃА рЃњрЃљрЃЋрЃџрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃў рЃњрЃљрЃЏрЃЮрЃЦрЃЋрЃћрЃДрЃюрЃћрЃЉрЃљ (deploy).|
