**Batching** рЃюрЃўрЃерЃюрЃљрЃЋрЃА РђърЃЊрЃљрЃЋрЃљрЃњрЃарЃЮрЃЋрЃЮрЃЌ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў рЃЊрЃљ рЃЏрЃћрЃарЃћ рЃћрЃарЃЌрЃўрЃљрЃюрЃљрЃЊ рЃЊрЃљрЃЋрЃљрЃЏрЃБрЃерЃљрЃЋрЃЮрЃЌРђю,  
**Streaming** рЃюрЃўрЃерЃюрЃљрЃЋрЃА РђърЃЏрЃЮрЃЋрЃўрЃЊрЃћрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃў Рђћ рЃЏрЃљрЃерЃўрЃюрЃЋрЃћ рЃЏрЃЮрЃЋрЃћрЃЏрЃАрЃљрЃ«рЃБрЃарЃЮрЃЌРђю.


|рЃЌрЃЋрЃўрЃАрЃћрЃЉрЃљ|**Batching**|**Streaming**|
|---|---|---|
|**рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃўрЃА рЃЊрЃарЃЮ**|рЃърЃћрЃарЃўрЃЮрЃЊрЃБрЃџрЃў (рЃЏрЃљрЃњ. рЃДрЃЮрЃЋрЃћрЃџ 5 рЃгрЃБрЃЌрЃерЃў, рЃАрЃљрЃљрЃЌрЃерЃў, рЃЊрЃдрЃћрЃерЃў)|рЃБрЃгрЃДрЃЋрЃћрЃбрЃў, рЃарЃћрЃљрЃџрЃБрЃа рЃЊрЃарЃЮрЃерЃў|
|**Latency (рЃЊрЃљрЃДрЃЮрЃЋрЃюрЃћрЃЉрЃљ)**|рЃЏрЃљрЃдрЃљрЃџрЃў РђЊ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃў рЃЏрЃБрЃерЃљрЃЋрЃЊрЃћрЃЉрЃљ рЃЊрЃљрЃњрЃЋрЃўрЃљрЃюрЃћрЃЉрЃўрЃЌ|рЃЊрЃљрЃЉрЃљрЃџрЃў РђЊ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃў рЃЏрЃБрЃерЃљрЃЋрЃЊрЃћрЃЉрЃљ рЃЏрЃЮрЃЋрЃџрЃўрЃАрЃЌрЃљрЃюрЃљрЃЋрЃћ|
|**рЃерЃћрЃАрЃљрЃцрЃћрЃарЃўрЃАрЃўрЃљ рЃарЃЮрЃфрЃљ...**|рЃЊрЃўрЃЊ рЃЏрЃљрЃАрЃљрЃџрЃљрЃќрЃћ aggregate рЃљрЃю reporting рЃњрЃГрЃўрЃарЃЊрЃћрЃЉрЃљ|рЃарЃћрЃљрЃџрЃБрЃа рЃЊрЃарЃЮрЃерЃў рЃарЃћрЃљрЃњрЃўрЃарЃћрЃЉрЃљ рЃњрЃГрЃўрЃарЃЊрЃћрЃЉрЃљ|
|**рЃўрЃюрЃцрЃарЃљрЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ**|рЃЏрЃљрЃарЃбрЃўрЃЋрЃў РђЊ SQL Jobs, Cron, ETL|рЃарЃЌрЃБрЃџрЃў РђЊ Kafka, RabbitMQ, EventHub|
|**рЃърЃарЃЮрЃфрЃћрЃАрЃўрЃА рЃЉрЃБрЃюрЃћрЃЉрЃљ**|Pull РђЊ рЃЏрЃўрЃЋрЃЊрЃўрЃЋрЃљрЃарЃЌ рЃЊрЃљ рЃЋрЃўрЃдрЃћрЃЉрЃЌ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃА|Push РђЊ рЃЏрЃЮрЃЋрЃџрЃћрЃюрЃљ рЃЌрЃЋрЃўрЃЌрЃЮрЃю рЃЏрЃЮрЃЊрЃўрЃА|
|**рЃЏрЃћрЃ«рЃАрЃўрЃћрЃарЃћрЃЉрЃљ / рЃњрЃљрЃЏрЃбрЃљрЃарЃЮрЃЉрЃљ**|рЃЮрЃърЃбрЃўрЃЏрЃўрЃќрЃћрЃЉрЃБрЃџрЃўрЃљ throughput-рЃќрЃћ|рЃЮрЃърЃбрЃўрЃЏрЃўрЃќрЃћрЃЉрЃБрЃџрЃўрЃљ latency-рЃќрЃћ|


#### ­ЪДЙ **Batch Processing**

**рЃЊрЃљрЃЏрЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃћрЃџрЃў рЃАрЃфрЃћрЃюрЃљрЃарЃћрЃЉрЃў:**

- рЃДрЃЮрЃЋрЃћрЃџрЃЊрЃдрЃўрЃБрЃарЃў рЃљрЃюрЃњрЃљрЃарЃўрЃерЃњрЃћрЃЉрЃљ (Daily Reports)
    
- Machine Learning рЃЏрЃЮрЃЊрЃћрЃџрЃўрЃА retraining рЃДрЃЮрЃЋрЃћрЃџ рЃдрЃљрЃЏрЃўрЃЌ
    
- Payroll / Billing рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ (рЃљрЃюрЃњрЃљрЃарЃўрЃерЃЊрЃћрЃЉрЃљ рЃЌрЃЋрЃћрЃерЃў рЃћрЃарЃЌрЃ«рЃћрЃџ)
    
- Data warehouse refresh (ETL Рєњ Data Lake)



#### РџА **Streaming Processing**

**рЃЊрЃљрЃЏрЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃћрЃџрЃў рЃАрЃфрЃћрЃюрЃљрЃарЃћрЃЉрЃў:**

- IoT рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃћрЃЉрЃўрЃА рЃбрЃћрЃџрЃћрЃЏрЃћрЃбрЃарЃўрЃљ (Smart Home, EV Charger)
    
- Fraud Detection рЃарЃћрЃљрЃџрЃБрЃа рЃЊрЃарЃЮрЃерЃў
    
- рЃЕрЃљрЃб рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃћрЃЉрЃў / WebSocket рЃерЃћрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃћрЃЉрЃў
    
- Stock price monitoring рЃљрЃю log analytics (ELK, Kafka)
